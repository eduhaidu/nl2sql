# ChatQL - A dynamic Natural Language to SQL System.

ChatQL allows you to communicate with any SQL database using plain english. All you have to do is write a request for your  database, and you will receive the desired results.

## Supported Database types

Currently, ChatQL supports the following database types:
    - SQLite
    - PostgreSQL
    - MySQL
    - SQL Server

## How it works

The system uses a pre-trained model (Phi 3.5) that takes the natural language question from the user and returns a SQL query that answers the user's question.
